# atmutm.online website project

This is a complete Vite + React website starter for your AURD / ATM / UTM showcase website.

## What is included

- Cranfield-style research website layout
- Sections for overview, media, case studies, outputs, and contact
- Ready to upload to GitHub
- Ready to deploy to Vercel after connecting the repository

## Run locally

1. Install Node.js
2. Open the project folder in Terminal
3. Run:

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Main files

- `src/App.jsx` — page content
- `src/styles.css` — page style
- `public/media/` — place your images and videos here

## Next edits you may want

- Replace placeholder text with official Cranfield / AURD wording
- Replace media placeholders with real images or embedded videos
- Add publications, team members, partner logos, and links
- Update contact email and project location

## Upload to GitHub

You can upload this whole folder to a new GitHub repository.

## Deploy to Vercel

1. Push this folder to GitHub
2. Log in to Vercel
3. Import the GitHub repository
4. Add your custom domain: `atmutm.online`
5. Configure DNS at your domain provider
